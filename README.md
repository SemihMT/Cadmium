# Cadmium - A hobby engine project
Cadmium is an engine I am writing to learn more about cmake, compilers, automated testing, emscripten and system/engine design.

## (a wishlist of) Features
- Cross platform between native platforms (windows/mac/linux) and web using Emscripten.
- General purpose
- Performant
- Easy scripting with Lua
- Renderer agnostic
- Unit tested using Catch2

## Current State

- [x] Milestone 1: Set up the build system, get a window rendering using one source file for both platforms.
- [ ] Milestone 2: Integrate Dear ImGui
- [ ] Milestone 3: Working Gameobject/Actor and component model
- [ ] Milestone 4: Lua integration + Lua API design
- [ ] Milestone 5: Making the engine renderer agnostic (with support for Vulkan, DX12, OpenGL, WebGPU, WebGL)
